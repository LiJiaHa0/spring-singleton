单例模式代码演示
单例模式：
一个类模板，在整个系统运行过程中，只允许产生一个实例（有且只有一个对象）new
例如：工厂模式中的配置文件，日历......。
实现单例模式的技术方案：
饿汉式、懒汉式、注册登记式、枚举式
饿汉式：
       在使用实例之前，不管你用不用，我都先new出来，避免了线程安全问题。
